Mosaic

Built for the Washington TSA Webmaster competition, Mosaic is a fictitious vegetarian restaurant with a goal to set the standard for sustainability in the restaurant industry, and was designed with these qualities in mind.

Project Overview

Features such as an add-to-cart functionality, a clear navigation path, and an embedded ingredients map benefit the user experience and make the website more effective. Design choices such as the color scheme, font, and images reflect the values of the restaurant while remaining modern and on-trend.
