Project Overview

Largely functionality based rather than design based. Uses the database software SQLite to store users and associated jobs. Uses PHP/Laravel to perform all backend functionality related to authentication, registration, and job editing. Uses HTML, blade components, and frameworks from Tailwind CSS to display all jobs.

TECHNOLOGY

PHP/Laravel with Herd, Blade components, SQLite, Tailwind CSS
